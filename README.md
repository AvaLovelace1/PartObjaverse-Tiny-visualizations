# PartObjaverse-Tiny visualizations

A webapp for visualizing meshes from the [PartObjaverse-Tiny](https://yhyang-myron.github.io/SAMPart3D-website/)
dataset (Yang et al., 2024).

PartObjaverse-Tiny is licenced under [CC-BY-NC-4.0](https://spdx.org/licenses/CC-BY-NC-4.0). I am not affiliated with
the creators of PartObjaverse-Tiny.

## Usage

Install [uv](https://docs.astral.sh/uv/), then run:

```
uv run streamlit run streamlit_app.py
```
